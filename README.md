* 📱 **Поиск по номеру телефона** — определение оператора, региона и привязок к аккаунтам.
* 👤 **Проверка никнеймов** — поиск совпадений никнейма в Telegram и десятках других сервисов.
* 📧 **Анализ email** — выявление связанных социальных сетей и публичных утечек.
